# AeroClean AI

Hackathon project combining AI, sustainability, and IoT concepts to optimize atmospheric water harvesting using smart fog nets.

## Features
- Fog and humidity prediction
- Smart deployment optimization
- Environmental monitoring dashboard
- Simulated IoT sensor integration
- Water collection and air quality analytics

## Tech Stack
- Python
- Machine Learning
- IoT Simulation
- Data Visualization

## Context
Developed during a hackathon at Sorbonne University Abu Dhabi.
